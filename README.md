# 快捷词典查询插件 (Quick Dictionary Search Extension) with Gemini AI

> 一个简单实用的浏览器扩展，让你在阅读文献或浏览网页时，可以快速查询选中的词汇。

这是一个为提升阅读效率而生的个人项目。通过在右键菜单中添加一个自定义选项，实现划词翻译和查询的功能。

## ✨ 主要功能 (Features)

- **划词即查**: 在任何网页上选中单词或短语。
- **右键菜单集成**: 通过右键菜单一键直达查询页面。
- **目标网站**: 默认使用[剑桥英语词典](https://dictionary.cambridge.org/)进行查询。
- **轻量高效**: 代码简洁，无后台资源占用，仅在需要时激活。

## 🚀 安装与使用 (Installation & Usage)

由于本插件尚未发布到应用商店，你需要通过开发者模式手动加载。

### 安装步骤

1.  **下载代码**: 将本仓库的代码下载到本地，或者使用 `git clone` 克隆。
2.  **打开浏览器扩展管理页面**:
    -   **Chrome**: 在地址栏输入 `chrome://extensions`
3.  **开启开发者模式**: 在页面右上角，找到并打开“开发者模式”的开关。
4.  **加载插件**: 点击左上角的“加载已解压的扩展程序”按钮，然后选择你刚刚下载并解压的项目文件夹。

### 使用方法

1.  在任意网页上，用鼠标选中一个你想要查询的英文单词。
2.  在该选中的文本上点击鼠标右键。
3.  在弹出的菜单中，选择 **"用剑桥词典查询 '...'"**。
4.  插件会自动在新标签页中打开查询结果页面。

## 🛠️ 技术栈 (Technology Stack)

- **核心逻辑**: `JavaScript` (ES6)
- **插件配置**: `manifest.json` (Manifest V3 规范)
- **开发环境**: Visual Studio Code

## 💡 进阶方向 (Future Plans)

这个项目还有很多可以探索和完善的地方，欢迎有兴趣的朋友一起贡献！

- [ ] **自定义词典**: 创建一个选项页面，允许用户自己设置查询的目标网址（例如：牛津词典、谷歌翻译、有道词典等）。需要用到 chrome.storage API来存储用户设置。
- [ ] **弹窗式查询**: 查询结果不再打开新标签页，而是在当前页面以小弹窗或侧边栏的形式展示。需要用到 Content Scripts 来向页面注入HTML和CSS，并可能需要从词典网站爬取数据（注意遵守网站的Robots协议）。
- [ ] **多词典支持**: 在右键菜单中创建一个父级菜单，可以同时选择在多个不同的词典中进行查询。
- [ ] **历史记录**: 记录用户的查询历史，方便回顾。
